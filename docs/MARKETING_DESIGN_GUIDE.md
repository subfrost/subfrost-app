# Subfrost Marketing Design Guide

## Product Overview

**Subfrost** is a Bitcoin DeFi platform that enables:
- **Multi-hop token swaps** on the Alkanes protocol
- **BTC wrapping** to frBTC (fractional BTC) for DeFi participation
- **Yield vaults** with DIESEL token incentives
- **Liquidity pool trading** with BUSD and frBTC bridges

**Key Value Propositions:**
1. Native Bitcoin DeFi without leaving the Bitcoin ecosystem
2. Multi-hop routing for optimal swap prices (via BUSD/frBTC bridges)
3. BTC yield generation through vaults (dxBTC strategy)
4. Governance through veDIESEL (vote-escrowed DIESEL)

**Target Audience:** Bitcoin holders who want to earn yield and participate in DeFi while maintaining Bitcoin exposure

## Current Brand Identity

### Visual Elements
- **Color Palette:** Ice/frost theme with blue gradients
  - Primary: `#284372` (deep blue)
  - Background gradient: `#CDE2FF` → `#F2F7FF` (light ice blue)

- **Effects:**
  - Glass morphism (frosted glass cards)
  - Animated snow particles
  - Subtle star/sparkle effects
  - Watermark background (subfrost branding)

- **Brand Assets:**
  - Snowflake logo/mark
  - "Subfrost" wordmark
  - Token icons (BTC, frBTC, DIESEL, BUSD)

### Design Aesthetic
- **Winter/frost theme** - clean, cool, professional
- **Minimalist** - focus on usability and clarity
- **Soft animations** - subtle movement without distraction
- **Glass elements** - transparency and depth

### 8. Brand Voice & Copywriting Guidelines

#### Tone
- **Professional but approachable** - Not too technical, not too casual
- **Confident** - "The Bitcoin DeFi Protocol" not "A Bitcoin DeFi Protocol"
- **Educational** - Explain complexity simply
- **Innovative** - Forward-thinking, cutting-edge

#### Words to Use
- Optimize, unlock, maximize, efficient, seamless, native, frost, winter-ready
- Multi-hop, yield, vault, liquidity, swap, bridge, route

#### Words to Avoid
- Complicated jargon without explanation
- Hype language (moon, rocket, etc.)
- Aggressive sales language
- Comparisons that diminish competitors (be competitive but respectful)

#### Example Copy Blocks

**Hero Headline Options:**
1. "The Bitcoin DeFi Protocol Built for Winter"

#### Midjourney/DALL-E Workflow
1. Generate initial concept art using prompts above

## Conclusion

Subfrost has a strong visual identity foundation with its winter/frost theme and ice blue color palette. The key to successful marketing design is **consistency** across all touchpoints while maintaining **clarity** for users navigating complex DeFi concepts.

Focus on:
1. **Visual hierarchy** - Guide eyes to key information
2. **Educational clarity** - Complex concepts explained simply
3. **Brand consistency** - Frost theme across all assets
4. **Performance** - Optimize for fast loading (web assets)
5. **Accessibility** - Ensure sufficient contrast, readable fonts

With AI design tools like Midjourney, DALL-E, Canva, and Adobe suite, you can rapidly generate high-quality marketing assets using the prompts and guidelines in this document. Start with the Phase 1 core assets and iterate based on community feedback and performance metrics.
